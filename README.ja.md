# Nab

C++23で実装した軽量なネットワークパケットキャプチャ・解析ツール。

## 概要

ネットワークの仕組みとモダンC++によるシステムプログラミングの実践を目的として作成した学習プロジェクトです。ネットワークトラフィックをリアルタイムにキャプチャし、柔軟なフィルタリング機能とともにパケット情報を表示します。

## 機能

- **パケットキャプチャ**：ネットワークインターフェースからパケットをキャプチャ
- **プロトコルフィルタ**：TCP、UDP、ICMP、IGMPでのフィルタリング
- **ポート・ホストフィルタ**：送受信先のIPアドレスやポートでのフィルタリング
- **サービス識別**：HTTP、HTTPS、DNS、SSHを識別
- **PCAPファイル出力**：Wireshark、`tcpdump` などでの解析のためにキャプチャを保存
- **グレースフルシャットダウン**：Ctrl+Cに対応して、統計情報を表示して終了

## 使用例

```bash
# 全トラフィックをキャプチャ
sudo ./nab

# TCPトラフィックのみをキャプチャ
sudo ./nab --protocol tcp

# DNSトラフィックのみをキャプチャ
sudo ./nab --port 53

# 特定ホストのトラフィックをキャプチャ
sudo ./nab --host 192.168.1.100

# HTTPSトラフィックをキャプチャしてファイルに保存
sudo ./nab --port 443 -o https_traffic.pcap
```

## ビルド

### 前提条件

- GCC 14以上・Clang 18以上などのC++23対応コンパイラ（GCC 14・15で動作確認済み）
- CMake 3.25以上
- Conan 2.x
- コマンドランナーの[just](https://github.com/casey/just)（任意）
- `libpcap` などのシステムパケットキャプチャライブラリ
- パケットキャプチャにはroot権限（`sudo`）が必要

### ビルド手順

`just` を使う場合：

```bash
just rebuild
just test
sudo just run
# 引数を渡す場合：sudo just run [引数]
```

手動でコマンドを実行する場合：

```bash
# 依存関係のインストールと設定
conan install . --output-folder=build --build=missing
cmake -B build -DCMAKE_TOOLCHAIN_FILE=build/conan_toolchain.cmake -DCMAKE_BUILD_TYPE=Release

# ビルド
cmake --build build

# テストの実行
ctest --test-dir build --output-on-failure

# プログラムの実行
sudo ./build/nab
```

## 技術的なポイント

- **モダンC++23**：`std::optional`、`std::ranges`、`std::print`、波括弧初期化、後置戻り値型、const correctnessなど
- **低レイヤのネットワーク処理**：EthernetおよびIPv4ヘッダを生バイトから直接パース
- **メモリ安全性**：ゼロコピーバッファアクセスのための `std::span` と `std::string_view`、RAIIのためのスマートポインタ
- **並行処理**：アトミック変数と条件変数によるスレッドセーフなパケット処理
- **テスト**：Catch2を用いたテストスイート（不完全なパケットや不正なデータなどのエッジケースを網羅）
- **モダンなツール**：Conanパッケージ管理、CMakeビルドシステム、`clang-tidy` 静的解析
